face_recognition - главная библиотека, она распознает лица на изображении и сравнивает их с шаблонами из БД
cv2 (OpenCV) - с ее помощью получаем изборажение с камеры (с веб и ip) и проходим по каждому кадру видео
bcrypt - используется для хэширования паролей (в БД хранятся хэшированные пароли, для безопасности), можешь сделать на этом акцент, https://knotes.ru/hes/ - общая инфа о хешах, https://pythonist.ru/heshirovanie-parolej-v-python-tutorial-po-bcrypt-s-primerami/ - статья которую я использовал, вот этот пример: hashAndSalt = bcrypt.hashpw(password.encode(), bcrypt.gensalt())
pymysql - используется для подключения к БД, я использовал MySQL
tkinter - использовали для создания клиентской части (форма авторизации)
